## About 

Application carried out in Rocketseat's Ignite course (React Native Path) with the aim of controlling a user's personal finances.

### Pre-Requirements

- React Native
- Node.JS
- CLI Expo 
- Internet access
- VSCode

### Installation

```
sudo yarn install (or npm i)
```

start the expo

```
sudo expo start
```
